  • Разработка на Django (MVT, ORM, формы, CBV)
  • Работа с PostgreSQL/MySQL (SQL запросы, индексы)
  • Покрытие кода тестами (unittest + pytest, >80% coverage)
  • Соблюдение PEP8, использование Git и GitHub
  • Безопасность: защита от XSS, CSRF, SQL-инъекций, настройка прав доступа.
  • Документация: код задокументирован, есть README с инструкцией по развертыванию.
