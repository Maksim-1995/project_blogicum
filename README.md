# Blogicum

Blogicum — это полнофункциональная блог-платформа, разработанная на Django.

## 🚀 Возможности

- Регистрация, авторизация, смена пароля (Django Authentication)
- Создание, редактирование, удаление постов (с поддержкой Markdown/CKEditor)
- Категории и теги для постов
- Комментарии к постам (с древовидной структурой)
- Поиск по постам и категориям
- Адаптивный дизайн (Bootstrap 5)
- Полное покрытие тестами (pytest)

## 🛠 Стек технологий

- **Backend:** Django 4.2, Python 3.10+
- **База данных:** SQLite для разработки
- **ORM:** Django ORM, оптимизированные запросы (select_related, prefetch_related)
- **Тестирование:** pytest
- **Frontend:** Bootstrap 5, FontAwesome, jQuery (для AJAX-комментариев)
- **Инструменты:** Git, pipenv/venv, pre-commit (flake8, black)
